## Prioritized Recommendations

1. Fix image extraction priority for EstateCV and Homes Casa Verde. Prefer full gallery/srcset candidates over thumbnail variants, and hard-scope Homes Casa Verde away from related/similar listing carousels before cover selection.
2. Fix area semantics for REMAX villas and Simply Cape Verde land/development listings. REMAX plot areas should not populate property_size_sqm as built area; Simply Cape Verde comma-thousands values such as 20,000 m2 must not parse as 20 m2, and land/development property type detection should be strengthened.
3. Add a public-feed dedupe review for the 6 cross-source clusters. The clusters are source-legitimate duplicates, but the feed currently publishes both sides without canonical consolidation.
4. Fix Ocean Property 24 title extraction/truncation so full source titles are surfaced.
5. Decide policy for source-authored weak titles and repeated development/unit titles. Cape Verde Property 24 generic titles and CCore/REMAX repeated source titles are not scraper errors, but they weaken marketplace UX.
6. Add a city recovery rule for Homes Casa Verde address blocks so Santa Maria is retained when the source page carries it.
